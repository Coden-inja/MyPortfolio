#Want to make Your Own Roasting AI Portfolio in Seconds ?

### Steps to Get Started

1. Clone the project  
2. Change your name  
3. Add your photos  
4. Add your project links  
5. Add your description  

---

Run the project locally

1. npm init -y
2. npm install express node-fetch dotenv
3. npm server.js
